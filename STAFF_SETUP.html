Living Culture Tools Setup

Use this guide to set up the Freight Costing app and browser helpers on a staff computer.

Install Scripts Checks How To Use Updates Troubleshooting

What Gets Installed

Freight Costing appStandalone app that calculates freight.
TampermonkeyChrome extension that runs the browser helpers.
LC FreightAdds the freight panel inside Cin7.
Quote Memo InfoAdds quote memo wording inside Cin7.
Install FeesAdds installation fee search and add tools inside Cin7.
Custom ProductsAdds custom product and pergola search inside Cin7.
Promo SummaryAdds current promotion summary search inside Cin7.
Copy SKUAdds the red SKU copy button on Living Culture product pages.

1. Install Chrome And Tampermonkey

  1. Open Google Chrome.
  2. Go to the Chrome Web Store.
  3. Search for Tampermonkey.
  4. Click Add to Chrome.
  5. Confirm the install.
  6. Open Chrome extensions with this address:
chrome://extensions
  1. Find Tampermonkey and make sure it is enabled.
  2. If Chrome shows Allow User Scripts, turn it on.

2. Install The Tampermonkey Scripts

Open each link below in Chrome. Tampermonkey should open an install page. Click Install or Update.

Cin7 LC Freight helper Cin7 Quote Memo Info helper Cin7 Installation Fee helper Cin7 Custom Product helper Cin7 Promo Summary helper Living Culture Copy SKU helper
After installing once, Tampermonkey checks GitHub for updates automatically.

3. Install The Freight Costing App

Mac

  1. Open the downloaded app file.
  2. Drag the app to Applications if prompted.
  3. Open Freight Costing.
  4. If Mac blocks it, right-click the app and choose Open.

Windows

  1. Open the installer supplied by Living Culture.
  2. Follow the prompts.
  3. Open Freight Costing.
Keep the Freight Costing app open while using the Cin7 LC Freight helper.

4. Check Everything Appears

Cin7

  1. Open Cin7 or Dear Systems in Chrome.
  2. Open a quote.
  3. Near Additional charges and services, check for:
Quote Memo Info LC Freight
  1. Near the top quote buttons, beside Scan, check for:
Install Fees Custom Products Promo Summary

Living Culture Product Pages

  1. Open a product page on livingculture.co.nz.
  2. Find the SKU near the product title.
  3. Check for the red Copy SKU button.
  4. Click it. The button should change to SKU copied.

How To Use The Tools

LC Freight

  1. Open a Cin7 quote.
  2. Make sure the Freight Costing app is open.
  3. Click LC Freight.
  4. Check the detected products, quantities, and address.
  5. Click Refresh freight with these quantities if needed.

Quote Memo Info

  1. Open a Cin7 quote.
  2. Click Quote Memo Info.
  3. Click Copy to copy text only.
  4. Click Copy + Fill Quote Memo to fill the quote memo and close the panel.

Install Fees

  1. Open a Cin7 quote.
  2. Click Install Fees.
  3. Search for the fee.
  4. Click Add.
Install fee data is loaded from the shared Google Sheet. If the sheet cannot be reached, the helper uses cached data or built-in fallback data.

Custom Products

  1. Open a Cin7 quote.
  2. Click Custom Products.
  3. Search for the custom product or pergola.
  4. Click Add.
Custom product data is loaded from the shared Google Drive file. If the file cannot be reached, the helper uses cached data or built-in fallback data.

Promo Summary

  1. Open Cin7.
  2. Click Promo Summary.
  3. Search or review current promotion details.
Promo summary data is loaded from the shared Google Sheet. If the sheet cannot be reached, the helper uses cached data or built-in fallback data.

Copy SKU

  1. Open a Living Culture product page.
  2. Click Copy SKU.
  3. Paste the SKU where needed.

Updating

Tampermonkey scripts update from GitHub automatically.

Manual update check

  1. Click the Tampermonkey icon in Chrome.
  2. Open Dashboard.
  3. Go to Utilities.
  4. Click Check for userscript updates.

For the Freight Costing app, install the newest app version supplied by Living Culture.

Troubleshooting

Buttons Do Not Show In Cin7

  1. Refresh the Cin7 page.
  2. Check Tampermonkey is enabled.
  3. Check the scripts are installed and enabled in the Tampermonkey dashboard.
  4. Make sure you are using Chrome.

A Script Link Shows 404 Or Does Not Open Tampermonkey

If the GitHub repo is private, staff must be signed into a GitHub account that can access it. For simple installs without GitHub login, the repo must be public or the script files need to be supplied another way.

LC Freight Does Not Load A Price

  1. Make sure the Freight Costing app is open.
  2. Check the quote has a product SKU.
  3. Check the quote has a shipping address.
  4. Close and reopen the LC Freight panel.

Install Fees Does Not Show Updated Sheet Data

  1. Check the computer has internet.
  2. Refresh Cin7.
  3. Open Install Fees again.
  4. Check the Google Sheet is shared so people with the link can view it.

Copy SKU Does Not Show On A Product Page

  1. Refresh the product page.
  2. Check the page has a visible SKU.
  3. Check the Living Culture Copy SKU script is enabled in Tampermonkey.
